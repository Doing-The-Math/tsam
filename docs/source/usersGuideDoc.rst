###########
User Guide
###########

You want to dig deeper into tsam, learn about the algorithms behind it and get and overview of the software structure of tsam?
Then the user guide will help you along with it. First, a short overview over the general procedure of time series clustering
for temporal aggregation based on `Hoffmann et al. (2020) <https://www.mdpi.com/1996-1073/13/3/641>`_ is given in the mathematical
description. The, the modules of the package are pesented with all their input and output options. Finally yet importantly, a
hyperlink to our tutorial on Github is provided.
